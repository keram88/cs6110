We encode the constraints for the puzzle as follows:\\
Each grid cell is an integer in the range 1 to $n$.\\
Use the Distinct function to make the values in the rows and columns distinct.\\
For each cage, we do:
\begin{itemize}
\item If the cell is a given, we assert that grid value is equal to the value.
\item If the cell is a sum, use Z3's Sum function to add the constraint that the cells in the cage add to the given value. Likewise for multiplication.
\item If the cell is a subtraction of cells $a$ and $b$, assert that either $a - b$ or $b - a$ is equal to the goal. Likewise for division.
\end{itemize}
